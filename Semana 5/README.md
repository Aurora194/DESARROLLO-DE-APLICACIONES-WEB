# Galería Interactiva con JavaScript

Este proyecto implementa una galería dinámica donde el usuario puede:

✔ Agregar imágenes mediante una URL
✔ Seleccionar una imagen
✔ Eliminar la imagen seleccionada


Usa DOM y eventos JavaScript.


## Tecnologías usadas


- HTML
- CSS
- JavaScript


## Publicación en GitHub Pages


1. Sube el proyecto a un repositorio
2. Ve a **Settings → Pages**
3. Selecciona la rama **main**
4. Guarda y abre el enlace generado
